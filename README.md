## 汎用CSVデータ可視化・分析ダッシュボード (DX-Data-Analyzer)

# 🌐 デプロイ・公開情報
本アプリは Render を利用してホスティングしており、以下のURLから実際に動作を確認いただけます。

デモサイトURL: https://data-analyzer-hx0h.onrender.com

# ⚠️ ご利用時の注意点
初回起動について: 無料プランのインスタンスを使用しているため、一定時間アクセスがないとサーバーがスリープ状態に入ります。URLをクリックしてから画面が表示されるまで、30秒〜1分ほど時間がかかる場合があります。

データの永続性: データベースには SQLite を使用しており、サーバーの再起動時にアップロードしたデータはリセットされます。動作確認の際は、お手持ちのCSVファイルをアップロードしてお試しください。

# 🚀 主な機能
・汎用CSVインポート

・項目数やヘッダー名が異なる任意のCSVファイルを自動解析して取り込み。

・JSON型データ構造により、データベース定義の変更なしで多種多様なデータに対応。

・データエクスプローラー (DataTables実装)

・全文検索・フィルタリング: キーワード入力によるリアルタイムな行抽出。

・列の並び替え (ColReorder): ヘッダーのドラッグ&ドロップによる表示順序の自由な変更。

・マルチソート: 各項目クリックによる昇順・降順の並べ替え。

・ページネーション: 大量データ表示時の表示件数切り替え。

・動的マルチグラフ分析 (Chart.js実装)

・自由な軸設定: 集計軸（ラベル）と数値項目（複数選択可）をユーザーが任意に指定。

・複数系列比較: 異なる数値項目（例：数量と単価）を同一グラフ上に並べて表示。

・多様なグラフ形式: 棒グラフ、折れ線グラフ、円グラフ、レーダーチャートに対応。

・バリデーション: 軸と数値の重複選択や、非数値データの自動除外・エラー回避処理。

# 📝レポート出力・エクスポート

画像保存: 生成したグラフをワンクリックでPNG画像としてダウンロード。

プリント最適化: グラフ表示画面の印刷・PDF出力時に不要な操作UIを自動非表示。

# 🛠 技術スタック
Backend: Python / Flask

Database: SQLite / SQLAlchemy (JSON型を活用した非構造化データ管理)

Frontend: HTML5 / CSS3 (Bootstrap 5) / JavaScript

Libraries: Pandas (データ処理), DataTables (表操作), Chart.js (グラフ描画)

Infrastructure: Render (Web Service / CD実装)

Deployment: GitHub Integration / Gunicorn
